History of F#

Originated from ML

Influenced by OCaml, C#, Haskell, etc.

Came out of Microsoft Research

Designed by Don Syme (@dsyme)

1.0 in 2005, 2.0 shipped with Visual Studio 2010

Latest version, 3.1.1, was an out of band release

Maintained by the F# Software Foundation (FSSF)

F# is

Statically and strongly typed

Functional first

Uses type inference

Open source (MIT License)

Targets CLR and Mono

Runs on Windows, OS X, Linux, iOS, Android, Windows Phone

Recursive functions


let rec factorial n =
    if n = 0 then 1
    else n * factorial (n-1)
                        

let rec gcd a b =
    if a = 0 then b
    else if a < b then gcd (b % a) a
    else gcd (a % b) b

Function currying

Every function takes one argument and returns one value


let f x y = ...

// val f : x:int -> y:int -> int

takes a parameter x that returns a functon that takes a parameter y


let f2 = f 2

// val f2 : (int -> int)

A record can be thought of as a simple immutable class, or a tuple with named fields

Pattern Matching

Match expressions


let rec factorial n =
    match n with
    | 0 -> 1
    | _ -> n * factorial (n-1)
                        

let rec sum list =
    match list with
    | [] -> 0
    | head :: tail -> head + sum tail

let rec length = function
    | [] -> 0
    | _ :: tail -> 1 + length tail
                        

let rec gcd a b =
    match a, b with
    | 0, _ -> b
    | _ when a < b -> gcd (b % a) a
    | _ -> gcd (a % b) b

Discriminated Unions


type Colour = | Red | Green | Blue

type Result =
    | Success
    | Error of string

type BinaryTree<'T> =
    | Leaf
    | Node of BinaryTree<'T> * 'T * BinaryTree<'T>

let rec length = function
    | Leaf -> 0
    | Node (left,_,right) -> 1 + (length left) + (length right)

Discriminated Unions


type Vector = Vector of float * float

let magnitude (Vector (x, y)) =
    sqrt (x*x + y*y)

magnitude (Vector (3.0, 4.0))
// type Vector = | Vector of float * float
// val magnitude : Vector -> float
// val it : float = 5.0

Pipelining

= Function Currying

+ Pipeline operator |>


let (|>) x f = f x
                        

sqrt 25.0
// is the same as
25.0 |> sqrt

Pipelining


List.map (fun x -> x * x) xs
// becomes
xs |> List.map (fun x -> x * x)

List.filter (fun x -> x > 0) xs
// becomes
xs |> List.filter (fun x -> x > 0)

Putting it together


xs
|> List.filter (fun x -> x > 0)
|> List.map (fun x -> x * x)
|> List.sum
|> sqrt
